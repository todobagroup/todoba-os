# Welcome

Welcome to the official documentation of **TODOBA**.

This documentation is the permanent knowledge base of the organization.

It preserves the identity, philosophy, architecture, culture, and long-term vision of TODOBA so that every future contributor—human or AI—can understand the organization before changing it.

This repository is not merely documentation.

It is part of the brain of TODOBA.

---

# What is TODOBA?

TODOBA is **not** a chatbot.

TODOBA is **not** a SaaS product.

TODOBA is **not** a collection of AI tools.

TODOBA is a **Living AI Organization**.

Its purpose is to continuously increase human capability through artificial intelligence while remaining aligned with human values.

---

# Core Mission

Increase human capability through AI so people have more time for meaningful things.

---

# North Star

Build an organization that continues learning, improving, and creating value long after its original creators are gone.
